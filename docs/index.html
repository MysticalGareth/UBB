What is this?

The Unstoppable Bitcoin Billboard (UBB) is an authorityless, censorship-resistant billboard that lives on Bitcoin.

How it works:

  • The billboard is a virtual canvas with 65,536 √ó 65,536 pixels
  • Anyone can claim a rectangular area (a "plot") with a Bitcoin transaction
  • Each plot contains an image and optionally a URL
  • Each plot is bound to a deed (a 600-sat UTXO)
  • Whoever controls the deed controls the plot
  • You can transfer, update, or retry your claim

Why UBB?

  • Supply-limited: The canvas is finite. First to claim wins.
  • Censorship-resistant: Once placed, plots are permanent.
  • No intermediaries: No tokens, no servers, no gatekeepers.
  • Fee market contribution: Every action generates Bitcoin transaction fees.
